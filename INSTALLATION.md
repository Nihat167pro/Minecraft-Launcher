# 📥 REALS Minecraft Launcher - Quraşdırma Bələdçisi\n\n**Bu bələdçi REALS Minecraft Launcher'ı Windows, macOS və Linux'a quraşdırmaq üçün addım-addım təlimatlar verir.**\n\n---\n\n## 📋 Sistem Tələbləri\n\n| Tələb | Minimum | Tövsiyə olunan |\n|------|---------|----------------|\n| **OS** | Windows 7+ | Windows 10+ / macOS 10.12+ / Linux |\n| **Java** | Java 8 | Java 11+ |\n| **RAM** | 2GB | 4GB+ |\n| **Disk** | 5GB | 10GB+ |\n| **İnternet** | Lazım | Tez xəttin tövsiyə olunur |\n\n---\n\n## 🔧 Quraşdırma Addımları\n\n### **1. Java Quraşdırın**\n\n#### Windows:\n```bash\n# Java 11+ versiyasını yükləyin:\n# https://www.oracle.com/java/technologies/downloads/\n\n# Yükləmə bitdikdən sonra kontrol edin:\njava -version\n```\n\n#### macOS:\n```bash\n# Homebrew ilə:\nbrew install openjdk@11\n\n# Kontrol:\njava -version\n```\n\n#### Linux (Ubuntu/Debian):\n```bash\nsudo apt update\nsudo apt install openjdk-11-jdk\n\n# Kontrol:\njava -version\n```\n\n---\n\n### **2. Layihəni Klonlayın**\n\n```bash\n# Depoyu klonlayın\ngit clone https://github.com/Nihat167pro/minecraft-launcher.git\ncd minecraft-launcher\n```\n\n---\n\n### **3. Maven ilə Quraşdırın**\n\n#### Maven Quraşdırın:\n\n**Windows:**\n```bash\n# https://maven.apache.org/download.cgi adresindən yükləyin\n# PATH-ə əlavə edin və kontrol edin:\nmvn -version\n```\n\n**macOS:**\n```bash\nbrew install maven\nmvn -version\n```\n\n**Linux:**\n```bash\nsudo apt install maven\nmvn -version\n```\n\n#### Layihəni Quraşdırın:\n\n```bash\n# Bütün asılılıqları yükləyin və quraşdırın\nmvn clean install\n\n# Yalnız paket yaradın (test keçməyin):\nmvn clean package -DskipTests\n```\n\n---\n\n### **4. Launcher'ı Işlədin**\n\n#### **Metod 1: JAR Faylı ilə**\n\n```bash\n# Quraşdırma bitdikdən sonra:\njava -jar target/reals-launcher-1.0.jar\n```\n\n#### **Metod 2: IDE ilə (IntelliJ IDEA / Eclipse)**\n\n1. Layihəni IDE-də açın\n2. `Main.java` sağ klikləyin\n3. \"Run\" seçin\n\n#### **Metod 3: Birbaşa Bash/Cmd ilə**\n\n```bash\nmvn exec:java -Dexec.mainClass=\"com.reals.launcher.Main\"\n```\n\n---\n\n## 🎮 İlk Başlanğıc\n\n1. **Launcher açılacaq** - Güzel GUI görsəniz, uğurlu quraşdırma keçdi! ✅\n2. **Minecraft Versiyas seçin** - 1.8.9 - 1.20.1 arasında seçin\n3. **Hesab seçin** - Oflayn ya da Microsoft Account\n4. **RAM ayarlayın** - Slider-i istifadə edərək (1GB - 16GB)\n5. **▶️ OYUNU BAŞLAT** - Düyməsinə klikləyin\n\n---\n\n## 🐛 Problem Həlli\n\n### **\"Java bulunamadı\" / \"Java not found\"**\n```bash\n# Java yolunu əl ilə göstərin:\n# Ayarlar → Java Yolu → C:\\Program Files\\Java\\jdk-11\\bin\\java.exe (Windows)\n# Ayarlar → Java Yolu → /usr/libexec/java_home (macOS)\n# Ayarlar → Java Yolu → /usr/bin/java (Linux)\n```\n\n### **\"Maven bulunamadı\" / \"Maven not found\"**\n```bash\n# Maven quraşdırın:\n# Windows: https://maven.apache.org/download.cgi\n# macOS: brew install maven\n# Linux: sudo apt install maven\n```\n\n### **\"Port 8080 istifadə olunur\" / \"Port 8080 in use\"**\n```bash\n# Fərqli port ilə başlayın:\njava -Dserver.port=8081 -jar target/reals-launcher-1.0.jar\n```\n\n### **Oyun başlamayır**\n```bash\n# Debug mode aktivləşdirin:\n# Ayarlar → Debug Mode → ON\n\n# Logs baxın:\ncat ~/.reals-launcher/launcher.log\n```\n\n### **Minecraft faylları yüklənmir**\n```bash\n# .minecraft qovluğu silin ve yenidən başlayın:\nrm -rf ~/.minecraft\n\n# Windows:\nrmdir /s %USERPROFILE%\\.minecraft\n```\n\n---\n\n## 📱 Executable (EXE) Yaradın (Windows)\n\n### **Launch4j ilə Çevirme:**\n\n```bash\n# 1.
