# 📝 REALS Minecraft Launcher - Dəyişiklik Günlüyü

## [1.0.0] - 2026-07-15

### ✨ İlk Buraxılış (Initial Release)

#### 🎮 Əsas Xüsusiyyətlər
- ✅ **Minecraft Versiyaları** - 1.8.9, 1.12.2, 1.16.5, 1.19.2, 1.20.1 dəstəyi
- ✅ **Account Management** - Microsoft Account və Oflayn hesab dəstəyi
- ✅ **Mod Manager** - Mod yüklə/sil/nizamla
- ✅ **Settings Panel** - RAM, JVM, dil ayarları
- ✅ **Multi-Language** - Azərbaycanca, English, Türkçe, Русский
- ✅ **GUI Interface** - Modern Dark Mode UI
- ✅ **Auto Updater** - Avtomatik versiya yoxlaması
- ✅ **Performance** - Parallel Downloads, Asset Cache

#### 🔧 Teknik Xüsusiyyətlər
- Maven 3.6+ ilə quraşdırma
- Java 11+ dəstəyi
- OkHttp3 ilə HTTP işləri
- GSON ilə JSON işləri
- SLF4J ilə logging

#### 📦 Daxil Olunular
- `reals-launcher-1.0.jar` - JAR Executable
- `INSTALLATION.md` - Detaylı Quraşdırma Bələdçisi
- `README.md` - Layihə Təsviri
- `.gitignore` - Git Ignore Konfiqurasiyası
- `pom.xml` - Maven Konfiqurasiyası
- Tam Java Kaynak Kodu (src/)

#### 🐛 Bilinən Məsələlər
- Microsoft OAuth tam inteqrasyon hazır deyil (v1.1-də)
- Multiplayer mod qurulması hazır deyil
- Profile switching beta mərhələsindədir

## Quraşdırma

### Tez Başlanğıc:
```bash
git clone https://github.com/Nihat167pro/minecraft-launcher.git
cd minecraft-launcher
mvn clean package -DskipTests
java -jar target/reals-launcher-1.0.jar
```

### Ətraflı Quraşdırma:
[INSTALLATION.md](INSTALLATION.md) sənədini oxuyun
